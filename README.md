## 🙌레시피 공유 사이트 '쉐프 노트'

## ❓'쉐프노트'는 뭐하는 사이트인가요?
- 나만의 레시피를 **공유**하고, 다른 쉐프님들의 레시피를 **탐색**할 수 있는 사이트입니다!

## 🧾주요 기능
- 레시피 조회
- 레시피 검색
- 레시피 등록
- 레시피 찜하기

## 🛠 전체 시스템 개요 

<details><summary>로그인 및 회원가입
</summary>

id, pw를 입력후, 로그인 버튼 push -> 로그인 성공!

❓계정이 등록 안되어있으면요?

로그인 페이지에 있는 **회원가입 버튼**을 눌러 회원가입 진행!

- id, pw, 닉네임을 작성 후 회원가입 버튼 push!
